## 🤖 SQL Table Generator AI
Uma ferramenta inteligente projetada para transformar descrições em linguagem natural em esquemas de banco de dados estruturados.
## ✨ Funcionalidades

* Prompt-to-SQL: Gere código DDL completo a partir de frases simples.
* Sugestão de Tipagem: Identifica automaticamente os melhores tipos de dados (INT, VARCHAR, JSON, etc.).
* Relacionamentos Inteligentes: Cria chaves estrangeiras e índices com base no contexto.
* Suporte Multi-Banco: Compatível com PostgreSQL, MySQL, SQLite e SQL Server.

## 🚀 Como usar

   1. Insira a descrição da sua tabela (ex: "Crie uma tabela de usuários com nome, email único e data de criação").
   2. Revise o esquema visual gerado pela IA.
   3. Copie o código SQL e execute no seu banco de dados.

## 🛠️ Tecnologias

* LLM: Gemini IA
* Backend: Python
